# DayPilot 项目简历

## 项目概述

**项目名称**：DayPilot（日程领航员）  
**项目类型**：Web前端应用  
**技术栈**：HTML5 + CSS3 + JavaScript（纯原生，无框架依赖）  
**代码规模**：约1200行（单文件应用）

---

## 核心功能

### 1. 多视图日历展示
- **日视图**：单日详细时间轴
- **周视图**：一周七天并排显示（默认视图）
- **月视图**：月度概览，显示每日任务摘要

### 2. 任务管理
- 创建/编辑/删除任务
- 任务属性：标题、项目、分类、日期、时间段
- 拖拽式时间选择（支持15分钟对齐）
- 实时时间线显示当前时间

### 3. 分类系统
- 预设分类：开发、会议、文档、代码检视、需求、设计、测试
- 自定义分类管理（新增/删除）
- 分类颜色标识
- 按分类筛选任务

### 4. 项目集管理
- 项目分组管理
- 按项目筛选任务
- 项目统计（任务数量）

### 5. 数据持久化
- **主方案**：File System Access API（文件系统直接读写JSON）
- **降级方案**：localStorage本地存储
- 数据文件：`sched_tasks.json`、`sched_cats.json`、`sched_projects.json`

### 6. 数据导出
- 支持导出为Markdown格式
- 按日期和项目分组
- 日期范围选择器（支持快捷选项：今天、本周、本月等）

## 技术亮点

### 1. 纯原生实现
- 零依赖，无需npm/webpack等构建工具
- 单HTML文件，开箱即用
- 体积小，加载快

### 2. 现代化UI设计
- 参考飞书日历的设计语言
- CSS变量管理主题色
- 流畅的CSS动画（fadeIn、slideUp、popIn等）
- 响应式布局（支持移动端）

### 3. 交互体验
- 拖拽创建任务（鼠标在日历区域拖动）
- 智能时间选择器Popover
- 键盘快捷键支持（ESC关闭弹窗）
- 平滑滚动和视图切换

### 4. 数据架构
```
任务(Task)
├── id: 唯一标识
├── project: 所属项目
├── title: 任务标题
├── categoryId: 分类ID
├── date: 日期（YYYY-MM-DD）
├── startMin: 开始时间（分钟数）
└── endMin: 结束时间（分钟数）

分类(Category)
├── id: 唯一标识
├── name: 分类名称
└── color: 颜色值

项目(Project)
├── id: 唯一标识
└── name: 项目名称
```

### 5. 核心算法
- **任务布局算法**：自动处理重叠任务的列排列
- **时间像素转换**：分钟数与像素位置的双向转换
- **日期计算**：周起始日、月份天数等日历逻辑

## 使用场景

1. **个人日程管理**：每日任务安排、时间规划
2. **项目进度跟踪**：按项目分类任务，查看项目进展
3. **工作记录**：导出日程为Markdown，用于工作汇报
4. **时间统计**：分析时间分配，优化工作效率
